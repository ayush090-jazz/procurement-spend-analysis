# Procurement Spend Analysis

This project analyzes procurement spend data to identify spending patterns across vendors, categories, departments, and time.

## Project Overview
- Cleaned and transformed raw procurement data using SQL
- Aggregated spend by vendor, category, department, and month
- Identified top vendors and major spending categories
- Created an interactive dashboard using Tableau

## Tools Used
- SQL (MySQL)
- Tableau
- CSV dataset

## Files Included
- consolidated_spend.csv : Cleaned and aggregated dataset
- procurement_spend_analysis.sql : SQL queries used for data cleaning and analysis

## Dashboard
The interactive dashboard was created using Tableau Desktop and includes:
- Vendor-wise spend analysis
- Category-wise spend distribution
- Monthly spend trends
- Filters for category, department, and time
